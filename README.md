d2h
===

`d2h` converts decimal numbers to hexadecimal numbers.

when you passed three decimal numbers, it will put a color code schema.


```shell
$ d2h 255 100
255 -> #ff
100 -> #64
```

```shell
$ d2h 255 100 0
#ff6400
```

## Install

```shell
$ cargo install --git https://github.com/funwarioisii/d2h.git
```
